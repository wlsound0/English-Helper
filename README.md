# 英语背诵辅助程序 - V0.1

## 项目结构

```
English/
├── append.py               # 添加单词模块
├── read_file.py            # 读取单词文件模块
├── main.py                 # 入口主文件
└── start.bat               # Windows启动脚本
```

## 运行方式

### 方法1：直接运行 main.py 文件
在编译软件中运行 `main.py`

### 方法2：使用启动脚本（Windows）
双击 `start.bat`

## 当前版本功能

- 检测单词文件完整性
- 主入口
- 添加/减小单个单词
- 输出全部单词
