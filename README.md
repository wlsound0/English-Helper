# 英语背诵辅助程序 - V1.3

## 注意事项
目前设置里单词本储存格式使用SQL模式没有任何实际作用。当然，是不会报错的！

## 项目结构

```
English/
├── Class/    
│   ├── Setting.py          # 设置类
│   ├── SettingOption.py    # 设置配置类
│	└── Word.py             # 单词、单词本类
├── dictionary/
│   ├── main.py             # 字典查询主函数
│   ├── stardict-ecdict-2.4.2.dict  # 字典数据文件
│   ├── stardict-ecdict-2.4.2.idx   # 字典索引文件
│	└── stardict-ecdict-2.4.2.ifo   # 字典索引信息文件
├── file/
│   ├── storage/
│   │   ├── Json.py         # 读取json文件
│   │   └── Txt.py          # 读取txt文件
│   ├── read_setting.py     # 读取设置文件
│	└── read_wordbook_txt.py    # 读取txt版的单词本文件
├── append.py               # 添加、查询单词模块
├── draw_word.py            # 抽取单词模块
├── main.py                 # 入口主文件
├── read_file_txt.py        # 读取单词文件模块
├── setting.py              # 设置模块
├── start.bat               # Windows启动脚本
└── upgrade_log.md          # 更新日志
```

## 运行方式

### 方法1：直接运行 main.py 文件
在编译软件中运行 `main.py`

### 方法2：使用启动脚本（Windows）
双击 `start.bat`

## 当前版本功能

- 检测单词文件完整性
- 主入口
- 添加/减小单个单词
- 批量添加/减小单词
- 输出全部单词
